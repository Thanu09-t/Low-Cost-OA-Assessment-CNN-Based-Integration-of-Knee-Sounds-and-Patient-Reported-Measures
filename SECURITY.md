# Security Policy

## Supported Versions

The following versions of **OA Insight** are currently supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0.0 | :x:                |

---

## Reporting a Vulnerability

We take the security of OA Insight seriously. If you find a security vulnerability, please report it via the contact below instead of opening a public issue.

Please send an email to **security@oainsight.com** containing:
1. A detailed description of the vulnerability.
2. Steps to reproduce or a Proof of Concept (PoC) script/request.
3. The potential impact of the issue.

We will acknowledge your report within 48 hours and work with you to coordinate a patch and release before public disclosure.
